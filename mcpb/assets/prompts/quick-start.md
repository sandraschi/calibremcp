# CalibreMCP Quick Start Guide

## Getting Started

Welcome to CalibreMCP! This guide will help you get up and running quickly with your Calibre ebook library management.

### First Steps

1. **Configure Library Path**: Set your Calibre library directory in the settings
2. **Test Connection**: Use the test connection tool to verify everything works
3. **Explore Your Library**: List your books and libraries to see what's available

### Essential Commands

- `list_libraries` - See all available Calibre libraries
- `list_books` - Browse your book collection
- `search_books` - Find specific books
- `get_book_details` - Get detailed information about a book

### Quick Tips

- Use Austrian efficiency - be specific about what you want
- Ask for recommendations based on your reading history
- Use series management tools to track your progress
- Leverage AI enhancements for better book discovery

## Troubleshooting

### Connection Issues
- Verify your Calibre library path is correct
- Ensure Calibre is running if using content server
- Check that metadata.db exists in your library directory

### Performance Issues
- Large libraries (10,000+ books) may take longer to process
- Use filtering options to narrow down searches
- Consider using specific library selection for faster results

### Getting Help
- Use the test connection tool to diagnose issues
- Check the library statistics for overview information
- Use the library health analysis for optimization suggestions
