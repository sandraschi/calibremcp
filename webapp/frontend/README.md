# Calibre Webapp Frontend

Next.js 15 frontend for Calibre library management.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:13001](http://localhost:13001) in your browser.

## Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:13000
```

## Build

```bash
npm run build
npm start
```
